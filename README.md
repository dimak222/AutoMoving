# AutoMoving

"AutoMoving" программа для автоматического перемещения мышки по бездействию через определённый интервал времени.

Что умеет:
1. Перемещать курсор мышки в случайном направлении на 1 пиксель по интервалу времени.
2. Отчёт времени идёт после последнего перемещения курсора (пользавателем или программно).
3. Опционально, отображать сдвиг или нет.
