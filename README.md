# AutoMoving

"AutoMoving" программа для автоматического перемещения мышки по бездействию через определённый интервал времени.

Что умеет:
1. Опционально, перемещать курсор мышки в случайном направлении на 5 пикселей по интервалу времени (Отчёт времени идёт после последнего перемещения курсора (пользавателем или программно)).
2. Опционально, отображать сдвиг или нет.
3. "Предотвращать сон" отдельной опцией.

P.S. Большая часть кода написанна с помощью вайб-кодинга.
